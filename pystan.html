Bayesian Inference to Predict CO2 Emissions with PyStan

Link to the GitHub

In this project, use Bayesian inference to predict CO2 emissions in the future. I take historical data of the CO2 emissions and then come up with a good functional form to describe the potential trend in the data. I use it as a likelihood function. I also come up with priors for the parameters of the model. Then I use PyStan to do Bayesian inference to get the posterior distribution using the data as well. I then make predictions about the future CO2 emission trends in the future.


The skills I used for this project

  • Programming with Python and using the PyStan library (thus Stan programming)
  • Bayesian inference
  • Working with data in Python using packages such as pandas and do preprocessing
  • Evaluation methods for model performance with tools such as residual, autocorrelation and pair plots.
  • Understanding of MCMC (Monte Carlo Markov Chain) sampling methods
