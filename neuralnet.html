Neural Network from Scratch

Link to the GitHub

In this project, I implemented a neural network from scratch using only the numpy package (No machine learning or statistics related packages were used and all of the algorithms are implemented by me). Then I tested my network on self generated data and the MNIST hand written digits dataset. The end product is a fully functioning neural network, suitable for classification problems. It uses sigmoid and softmax activation functions and cross-entrpy loss. I aimed for the network to be very flexible. The users can add any number of layers and choose any number of nodes for each layer and the forward and backpropogation work without any issues. The syntax I used for the functions is inspired ny the Keras library and you will see how simple and intuitive it is to use it.


The skills I used for this project

  • Clean, object oriented programming with Python
  • Deep understanding of neural networks including activation functions, forward propogation, backpropogation, gradient descent
  • Linear Algebra and Calculus for all of the operations with the weight adjustments, gradient descent algorithm etc.
  • Algorithms to implement gradient descent, backpropogation etc.
  • Understanding of data science practices for training and testing the model with data and checking baselines.
  • Complexity theory for testing the runtimes
