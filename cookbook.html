CookBook: A Social Network for Food

Link to the GitHub

CookBook is a platform for personal recipe sharing. It includes all the functionality of a social media app, but it is for food.

The idea for CookBook started off as a recipe preservation project. The diversity of recipes is immense, whether it be due country to country, person to person or region to region differences. However, the recipes that are easily accessible through the internet or even recipe books are standardized versions of these recipes.

Each recipe is unique with a personal or historical story behind it. Usually a generational transfer is the way recipes and their stories are passed on, yet this isn't always very effective and recipes disappear all the time.

I want CookBook to become our collective memory of recipes; a platform where we can share our recipes, access other people's recipes and collaborate on CookBooks.

The skills I used for this project

  • Django for the server
  • Django ORM for the database (with an SQLite database)
  • HTML, CSS with Bootstrap for frontend
  • Object Oriented Programming
  • User authentication with multiple users and different hierarchical levels of users
  • HTTP protocol, POST, GET requests
  • Agile software development. I conducted several rounds of interviews (verbal and interactive) aiming to understand the needs and expectations of the users and reiterated on the product so that it serves those needs and expectations better. At the same time time I kept the architecture of the project very modular. Even though it was a solo project I had everything organized and I could concentrate on a single concern at a time.
