Computer Vision: Facemask Detection

Link to the GitHub

In this project I built a convolutional neural network with the keras library to build a model to detect if a person is wearing a facemask or not. I build a baseline model and then experiment with different hyperparameter settings to fine tune my model including different activation functions, different optimizers and different dropout rates. After the fine tuning I got a 95% accuracy with my test data. I then build a clone of the VGG16 model and train it with the same dataset and compare the performances of the two models.


The skills I used for this project

  • Understanding of convolutional neural networks
  • Building machine learning models and training them
  • Ability to experiment and fine tune a CNN
  • Using the Keras library with TensorFlow backend
  • Image data manipulation
  • Cloning state of the art models
