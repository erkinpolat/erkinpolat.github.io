NP and NP-Complete with Phutball and Sudoku


This was initially a school project that I did, which later on I turned into a medium article. It is about the theory of complexity. It explains how the 3-SAT problem is reducible to the Phutball game, how Sudoku is reducible into the SAT problem and how therefore by solving the Phutball game we can solve Sudoku puzzles. I provide proofs, language definitions and Turing machines along the way to show how the whole process works out. I also wrote my own polynomial time reduction function from Sudoku to SAT in Python and included it at the end of the article.
